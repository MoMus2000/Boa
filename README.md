# Boa

Creating a language in Python.

```bash
# Load source code
python3 main.py path_to_source.boa

# Launch interpreter
python3 main.py

# Run tests
./tests.sh
```

Sample Boa Code

```lua
var hello = "Hello";
var world = "World";

print hello + world;

{
    var boa_lang = 1;
    boa_lang = boa_lang + 1;
    print boa_lang;
}
```
